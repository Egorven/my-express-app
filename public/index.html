Список любимых песен

Это клиентская страница. API доступно по /api/songs.

Как использовать API:

  • GET /api/songs — получить все песни
  • GET /api/songs?genre=Rock — фильтр по жанру
  • POST /api/songs — добавить песню

Пример POST-запроса (JSON):

{
  "title": "Hotel California",
  "artist": "Eagles",
  "genre": "Rock"
}
